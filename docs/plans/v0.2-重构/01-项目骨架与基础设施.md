### 3.2 配置加载（`internal/infra/config/`）

**配置文件结构**（`config/config.yml`）:

```yaml
server:
  port: 8080
  mode: development      # development / production

database:
  host: localhost
  port: 3306
  name: gantt_saas
  user: root
  password: ""
  max_open_conns: 50
  max_idle_conns: 10
  conn_max_lifetime: 3600  # 秒

  addr: localhost:6379
  password: ""
  db: 0

  level: info             # debug / info / warn / error
  format: json            # json / console
  output: stdout          # stdout / file path

  enabled: false          # 全局 AI 开关
```

**加载方式**: Viper 读取 YAML → 绑定到 Go struct → 支持环境变量覆盖（`GANTT_DATABASE_HOST` 等）。

### 3.3 数据库初始化（`internal/infra/database/`）

```go
// database.go
func NewDB(cfg *config.DatabaseConfig) (*gorm.DB, error) {
    dsn := fmt.Sprintf("%s:%s@tcp(%s:%d)/%s?charset=utf8mb4&parseTime=True&loc=Local",
        cfg.User, cfg.Password, cfg.Host, cfg.Port, cfg.Name)
    db, err := gorm.Open(mysql.Open(dsn), &gorm.Config{
        Logger: newGormLogger(),
    })
    // 连接池配置
    sqlDB, _ := db.DB()
    sqlDB.SetMaxOpenConns(cfg.MaxOpenConns)
    sqlDB.SetMaxIdleConns(cfg.MaxIdleConns)
    sqlDB.SetConnMaxLifetime(time.Duration(cfg.ConnMaxLifetime) * time.Second)
    return db, err
}
```

### 3.4 Redis 初始化（`internal/infra/cache/`）

```go
// cache.go
func NewRedis(cfg *config.RedisConfig) *redis.Client {
    return redis.NewClient(&redis.Options{

        Password: cfg.Password,
        DB:       cfg.DB,
    })
}
```

```go
    r := chi.NewRouter()
    r.Use(middleware.RequestID)

        // M04 挂载: r.Mount("/employees", employeeRouter)
```

### 3.7 入口文件（历史单体入口，现已废弃）

### 3.8 Docker Compose（开发环境）

```yaml
# deploy/docker-compose.yml
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: gantt_saas
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

  app:
    build: .
    ports:
      - "8080:8080"
    depends_on:
      - mysql
      - redis
    environment:
      GANTT_DATABASE_HOST: mysql
      GANTT_REDIS_ADDR: redis:6379

volumes:
  mysql_data:
```

## 五、约定规范

### 5.1 代码分层（每个业务模块统一遵循）

```
internal/core/employee/
├── model.go          # 领域模型（GORM Model）
├── repository.go     # 数据访问（interface + 实现）
├── service.go        # 业务逻辑
├── handler.go        # HTTP Handler（chi）
└── routes.go         # 路由注册
```

### 5.2 错误处理约定

```go
// 统一错误响应
type ErrorResponse struct {
    Error struct {
        Code    string `json:"code"`
        Message string `json:"message"`
    } `json:"error"`
}
```

### 5.3 分页约定

```go
// 请求: ?page=1&size=20
// 响应:
type PageResponse struct {
    Data  interface{} `json:"data"`
    Total int64       `json:"total"`
    Page  int         `json:"page"`
    Size  int         `json:"size"`
}
```
