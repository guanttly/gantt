# 导航栏布局优化说明

## 问题诊断

### 修改前的问题：

```
[图标] 排班工作台    [图标] 数据管理
  ↓                    ↓
水平排列导致：
1. 占用空间过宽
2. 文字容易换行
3. 视觉不够整齐
```

### 修改后的布局：

```
  [图标]      [图标]
排班工作台    数据管理
  ━━━
 (激活态)
  ↓
垂直排列优势：
1. 节省水平空间
2. 视觉更整齐
3. 类似现代 Tab 设计
```

## 具体修改

### 1. 布局方向改变

**之前：** `flex-direction: row` (水平)
**现在：** `flex-direction: column` (垂直)

### 2. 尺寸优化

- **图标**: 18px → 20px (更突出)
- **文字**: 14px → 13px (更紧凑)
- **最小宽度**: 无 → 80px (统一宽度)
- **间距**: gap 8px → 4px (图标和文字更紧凑)

### 3. 激活态指示器

- **位置**: bottom: -1px → bottom: 0 (在底部边缘)
- **宽度**: 24px → 32px (更明显)
- **样式**: 完整圆角 → 顶部圆角 (更像标签页)

### 4. 响应式优化

```scss
// 平板 (≤768px)
- Logo 文字正常显示
- 导航项紧凑布局
- 图标 20px → 18px
- 文字 13px → 11px

// 手机 (≤480px)
- Logo 只显示图标
- 导航项极简布局
- 文字 10px
```

## 视觉效果

### 布局结构

```
┌────────────────────────────────────────────────────┐
│  📅 智能排班     [📅]          [⚙️]          👤    │
│                排班工作台     数据管理               │
│                 ━━━━━                              │
└────────────────────────────────────────────────────┘
```

### 导航项状态

#### 正常态

```
  [📅]
排班工作台
```

#### 悬停态 (hover)

```
  [📅]
排班工作台  ← 浅蓝色背景，蓝色文字
```

#### 激活态 (active)

```
  [📅]
排班工作台  ← 浅蓝色背景，蓝色文字，加粗
  ━━━━━    ← 底部蓝色指示条
```

## CSS 变量使用

```scss
// 间距
--spacing-xs: 4px
--spacing-sm: 8px
--spacing-base: 16px
--spacing-lg: 20px
--spacing-xl: 24px

// 颜色
--text-regular: #606266      // 默认文字
--primary-color: #367AC2     // 激活/悬停
--primary-light: #004ea21a   // 激活背景
--bg-hover: #f5f7fa          // 悬停背景

// 字体
--font-size-sm: 13px
--font-weight-medium: 500
--font-weight-semibold: 600
```
